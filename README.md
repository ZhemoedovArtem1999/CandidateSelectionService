# CandidateSelectionService
## Работа с кандидатами
- Добавить кандидата
- Редактировать кандидата
- Получить кандидатов по фильтру с пагинацией
- Перевести кандидата в сотрудники
## Работа с поиском
- Поиск кандидатов и сотрудников по ФИ
- Запись в соответствующие таблицы БД
- Отправка в группу посредством телеграм бота
### Куда отправляется
- Группа https://t.me/group_hr_on_bot
- Бот @CandidateSelectionServiceBot

## Стек:
- .Net 8
- C#
- EF
- PostgreSQL
